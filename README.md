# Number Guessing Game

A simple number guessing game built using Python.

The program randomly generates a number between 1 and 10, and the user has 3 attempts to guess the correct number.

## Features

- Random number generation
- Maximum 3 attempts
- Hint system:
  - "Too high"
  - "Too low"
- Displays remaining attempts
- Shows total attempts if guessed correctly
- Reveals correct number when game is over

## Concepts Used

- Functions
- Loops
- Conditional statements
- Random module
- User input handling
- Counter variables

## Technologies

- Python 3
